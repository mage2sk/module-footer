# Changelog

All notable changes to this extension are documented here. The format
is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.5]

### Changed
- Code cleanup: removed redundant inline comments and docblocks from the PHP source. No functional changes.

## [1.0.0] - Initial release

### Added
- **Multi-column footer layout** — configurable 2, 3, or 4 column
  footer with independent content per column.
- **Column 1 — Logo and About** — store logo, about text, and social
  media icons.
- **Columns 2 and 3 — Link columns** — JSON-configured navigation
  links with admin JSON beautifier (beautify, minify, validate).
- **Column 4 — Contact information** — phone (click-to-call), email
  (mailto), physical address, and working hours.
- **Social media icons** — inline SVG icons for Facebook, Twitter/X,
  Instagram, LinkedIn, YouTube, and Pinterest.
- **Newsletter section** — AJAX subscription form with configurable
  title, subtitle, benefits list, placeholder, and button text.
- **Back to top button** — smooth-scroll button with configurable
  position (bottom-right or bottom-left), appears after 300px scroll.
- **Footer bottom bar** — copyright text with `{{year}}` auto-replace,
  payment icons (Visa, Mastercard, PayPal, Amex), and configurable
  bottom links.
- **Dual theme support** — ships both Hyva (Alpine.js) and Luma
  (vanilla JS) templates; correct set loads automatically.
- **Theme Customizer integration** — all footer, newsletter, and
  back-to-top colors are read from the Theme Customizer config.
- **Full admin configuration** under Stores > Configuration > Panth
  Extensions > Footer Configuration.

### Compatibility
- Magento Open Source / Commerce / Cloud 2.4.4 - 2.4.8
- PHP 8.1, 8.2, 8.3, 8.4
- Hyva Theme and Luma Theme
